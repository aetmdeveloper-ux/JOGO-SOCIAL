# 🤖 MANDROID: O Guardião das Redes

**Desenvolvido por MANDROIDAPP DEVELOPER · Adão Everton Tavares Miranda**

---

## 📋 Sobre o Jogo

MANDROID é um jogo de exploração e ação baseado em browser, onde você controla um robô futurista que deve salvar a Social City de um vírus digital.

**Gênero:** Action/Exploration  
**Controles:** Somente Mouse  
**Plataforma:** Browser (HTML5 Canvas)

---

## 🎮 Como Jogar

| Ação | Controle |
|------|----------|
| Mover personagem | Mover o mouse |
| Atacar inimigos | Clique Esquerdo |
| Interagir com NPC | Clique quando estiver perto |
| Avançar diálogo | Clique em qualquer lugar |

---

## 📁 Estrutura de Arquivos

```
mandroid/
├── index.html       ← Arquivo principal
├── style.css        ← Estilos e design
├── script.js        ← Lógica completa do jogo
└── assets/
    ├── robo.png     ← Personagem Mandroid (você)
    ├── face.png     ← Facebook Mascote (NPC)
    ├── insta.png    ← Instagram Mascote (NPC)
    ├── whats.png    ← WhatsApp Mascote (NPC)
    ├── tiktok.png   ← TikTok Mascote (NPC)
    └── cidade.png   ← Cenário da Social City
```

---

## 🖼️ Adicionando as Imagens

Coloque os arquivos de imagem na pasta `assets/` com os nomes exatos:

- `robo.png` — O robô Mandroid (personagem principal)
- `face.png` — Facebook Mascote
- `insta.png` — Instagram Mascote
- `whats.png` — WhatsApp Mascote
- `tiktok.png` — TikTok Mascote
- `cidade.png` — Imagem de fundo da cidade

> **Nota:** O jogo funciona mesmo sem as imagens! Serão usados sprites desenhados proceduralmente como fallback.

## 🗺️ Fases do Jogo

---

## 👾 Inimigos

- **Glitch** (roxo/pink) — Movimento errático e imprevisível
- **Sombra** (roxo escuro) — Forma nebulosa e veloz
- **Bot** (ciano) — Mecânico e resistente
- **Vírus Central** (boss) — 300 HP, spawna minions, dispara projéteis!

## 🚀 Deploy no GitHub Pages

1. Faça upload de todos os arquivos para um repositório no GitHub
2. Vá em **Settings → Pages**
3. Selecione a branch `main` como source
4. Acesse: `https://seu-usuario.github.io/nome-do-repo`
